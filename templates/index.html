Email Provider Api

This API is designed and coded to return messages from provided email account. As of now, in test mode, only email given for test mode will work with this API.

Gmail Provider

  • Go to http://37.221.193.135/auth to give permission to this app to access your Gmail
  • Go to http://37.221.193.135/example@gmail.com to get example@gmail.com mails {id, subject, date and snippet}
  • Go to http://37.221.193.135/example@gmail.com/id to get full message
  • Go to http://37.221.193.135/example@gmail.com/number to get particular numbers of mails. In case example@gmail.com mails are much, replace number with a number e.g 10. This will return full 10 mails.
  • Go to http://37.221.193.135/example@gmail.com/full to get full mails in user account. Hence, this can take a long time due to emails available in user account. This will return {label(SPAM, INBOX, SENT, DRAFT), subject, date, snippet, message}
